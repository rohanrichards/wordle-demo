# Wordle Clone

A scaffold for building a Wordle clone in the browser. Built with Vite, React 19, and TypeScript, styled with Tailwind v4, tested with Vitest and React Testing Library.

## Commands

- `pnpm dev` — start the dev server
- `pnpm test` — run tests in watch mode (`pnpm test run` for one-shot)
- `pnpm build` — production build
