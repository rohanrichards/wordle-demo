# Wordle Clone

A scaffold for building a Wordle clone in the browser.

## Stack
- Vite + React 19 + TypeScript
- Tailwind v4 for styling
- Vitest + React Testing Library for tests

## Commands
- `pnpm dev` — start dev server
- `pnpm test` — run tests in watch mode (`pnpm test run` for one-shot)
- `pnpm build` — production build
